# Ex Friend - 情感陪伴技能

[English](./README_EN.md) | 中文

## 简介

Ex Friend 是一款情感陪伴工具，通过分析用户上传的聊天记录，提取伴侣的语言风格、语气习惯和情感表达方式，在用户需要时以"伴侣"的身份与其交流。

## 适用场景

- 用户失恋或分手
- 伴侣离世（丧偶）
- 用户主动要求复现某人语气进行情感陪伴

## 功能特性

### 📁 支持多种文件格式
- 文本文件（.txt）
- 网页文件（.html）
- JSON 文件（.json）
- Word 文档（.docx）
- PDF 文件（.pdf）
- 音频文件（.mp3/.amr/.wav）

### 💬 智能聊天
- 自动分析聊天记录，提取人格特征
- 以伴侣的语气和风格与用户聊天
- 保持语言风格一致性

### 💙 安慰消息
- 根据伴侣风格生成安慰话语
- 支持定时发送

### 📖 回忆过去
- 提取特定时期的美好对话
- 支持指定时间范围回忆

### 🎤 语音功能
- 首次使用提供 3 种音色选择
- 支持文字转语音（需要用户明确要求）

## 隐私保护

⚠️ **重要提示**
- 所有聊天记录在本地处理，不上传至云端
- 不会保存用户的真实个人信息
- 仅用于情感支持，不用于任何形式的欺骗

3. 确认伴侣角色
4. 开始情感陪伴聊天
